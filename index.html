hidten

A framework for combining hidden Markov- and related models with modern deep learning.

Find the documentation here.

Summary

  • build models over a discrete latent state space

  • currently: Hidden Markov Models (HMMs), future: Conditional Random Fields (CRFs)

  • gradient-based and highly parallel training

  • highly parallel inference

  • can be part of larger deep learning models

  • pythonic, clean, extensible, tested

  • future (?): swapable machine learning backend (currently only TensorFlow)

Installation

git clone https://github.com/Gaius-Augustus/hidten

For users:

pip install -e .[tensorflow]

For developers:

pip install -e .[tensorflow,test,docs]

Build the docs:

cd docs && make html
